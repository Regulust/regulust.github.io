LaTex 的一点入门经验 & TeX Live 安装

2017-03-25
LaTex, Tips, 技术

学习使用LaTex在入门时确实花费了不少时间，这里把自己的一点入门经验分享一下～

  • 首先需要先了解一下什么是LaTex，latex吧等一些社区，可以获取一些简单的关于LaTex的消息，另外这里还有一份LaTex简单的入门，可以先大概看看第一章基本知识了解一下一份不太简短的LATEX介绍；
  • LaTex有许多发行版，包括CTex、TexLive等，这里推荐开源且更加全面的TeXLive (TeXLive 2016发行版的下载) ，网页中的链接都可以下载，不过推荐通过ctan的镜像站由迅雷下载（如下图），整个套装接近3G，教育网会有有几MB/s的速度，下载比较快； 安装过程可以参考：TeX Live 2014安装引导流程以及Linux 系统下原版 TeX Live 2016 的安装与配置。

附图：

  • TeXLive只是LaTex排版系统的编译部分，LaTex支持多种编辑器来编辑，包括TeXworks、TexStudio、Texmaker等，这里并不推荐与ctex整合的winedit（实为非开源收费软件破解），作为新手，推荐使用Texmaker，界面友好，直观简单，熟悉之后推荐转为TeXstudio …
    另外，LaTex也有一些在线编辑器，好处是只要有网就可以编辑，并且同步不会丢失，在线模板也比较多，这里有两个推荐的在线LaTex编辑网站：TeXIDE (中文)、ShareLaTex(英文)
