尝试在Ubuntu环境搭建Hexo及添加图片

2017-03-18
  • 技术

晚上本来还打算做些“正事”，修正之前的计算结果（最后也没修正对），浏览网页，偶然看到一篇在hexo-blog中添加图片的帖子：“hexo博客进阶－相册和独立域名”，突然想到之前搭的Hueman主题的帖子没有标题图不好看，所以一时兴起想照着教程把标题图加上～然而问题来了，现在的工作环境是Ubuntu，原来使用的Windows搭建的hexo，现在先得在Ubuntu中把Hexo搭起来…

本来应该挺简单的事情，也是折腾了个把小时，主要参考了“手把手教你在Ubuntu平台搭建 Github + Hexo 免费博客”这篇帖子，但在安装node.js时，没有使用文中的方法（编译时间有点久…），而是参考了“Ubuntu 16.04 安装 nodejs”这篇帖子，最后成功安装了node 7.6,(其实还是想要6.10LTS来着)，然后也参考了“Hexo官方文档”，完成了Hexo的安装，安装中注意某些步骤可能需要sudo权限…

接下来，开始试着给blog添加图片吧…转眼亥时过半，今天的任务又完不成了…github知识库改造开始～

……

咳咳，怪我没仔细看看帖子和blog…github知识库有容量限额，所以…临时帖子贴图可以，大量图片、相册还得是“七牛云”等…

ps：对了，今天晚上还收到了红鸭梨的好消息，他复试过了，还真是在追随王校长的道路上越走越远，哈哈哈，恭喜他～
