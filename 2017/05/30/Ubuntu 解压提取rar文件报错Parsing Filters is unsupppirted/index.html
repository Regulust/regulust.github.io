Ubuntu解压提取rar文件报错:parsing filter is unsupported

2017-05-30
|   Tips, 技术
阅读次数: 1000000
本文字数: 67 阅读时长: 1 分钟

同学发来一个.rar压缩包，Ubuntu16.10 右键提取却出现了"Parsing filter is unsupported"报错

上网一查，原来缺"unrar"解压命令…
终端输入sudo apt install unrar安装unrar即可～
