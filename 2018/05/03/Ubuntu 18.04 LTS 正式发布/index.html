Ubuntu 18.04 LTS 正式发布

2018-05-03
|   News
阅读次数: 1000000
本文字数: 570 阅读时长: 2 分钟
  1. 主要新特性：
    1. Linux kernel 升级为 v4.15
    2. OpenJDK
    3. 安全性改进
    4. 系统更新
    5. Ubuntu 桌面
    6. Ubuntu Server

参考转载自：开源咨询

2018年4月26日，Canonical在英国伦敦正式发布Ubuntu 18.04 LTS.
Ubuntu 18.04 为LTS长期支持版，版本代号Bionic Beaver(仿生海狸)。正如他的代号，这次更新专门针对Multi-Cloud基础设施，机器学习，人工智能，软件开发而优化。

主要新特性：

Linux kernel 升级为 v4.15

  • cgroup v2
  • AMD 安全内存加密
  • 最新 MD 驱动 (RAID 加强)
  • 针对 SATA Link 电源管理的改进

OpenJDK

  • Ubuntu 18.04 目前默认采用的 JRE/JDK 是 OpenJDK 10 。等到 Oracle 在 9 月发布 OpenJDK 11 后，18.04 还将同步更新。

安全性改进

  • Ubuntu 18.04 LTS 将 gcc 设为默认的编译应用程序，以更有效地使用地址空间布局随机化（ASLR）。除了少数例外情况外，所有主要软件包都已重新构建，以利用此功能。

系统更新

-Python 2 不再默认安装。它仍然包含在内，但 Ubuntu 18.04 将是包含 Python 2 的最后一个 LTS 。Python 3 已更新至 3.6 。

Ubuntu 桌面

  • Ubuntu 18.04 LTS 默认的显示服务采用的是 X.Org 。之前呼声很高的 Wayland 会作为一个Technical Preview 提供服务（可在登录界面上进行切换选择），并有望在 20.04 LTS 中成为默认显示服务。
  • 提供最小化安装选项
  • GNOME 3.28
  • LibreOffice 已更新至 6.0
  • 表情符号现在在大多数应用中已经可以以彩色显示
  • 日历现在支持天气预报
  • spice-vdagent 已预先安装，以便为 Spice 客户端提供更好的性能
  • gconf 不再默认安装，因为已被 gsettings 取代

Ubuntu Server

  • 采用下一代 subiquity Ubuntu Server Installer
  • 弃用 ifupdown
  • LXD 3.0
  • QEMU 2.11.1
  • libvirt 4.0
  • DPDK 17.11.x
  • Open vSwitch 2.9
  • chrony 取代 ntpd 作为推荐的 NTP 协议服务器
  • cloud-init 18.2
  • curtin 18.1
  • MAAS 2.4b2
  • SSSD 1.16.x
  • Nginx 1.14.0
  • PHP 7.2.x
  • Apache 2.4.29
