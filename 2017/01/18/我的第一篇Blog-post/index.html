我的第一篇Blog post

2017-01-18
  • 日记
  • 随笔


这是我在github上的第一篇blog post，折腾了一天…

从漫无目的的搜索，到又浏览了码云的code托管平台，再到选定Hexo（码云支持1000个库，并且可以设置为private，但可惜不能正常显示hexo输出的网页）；然后又图颜值，搜索好看的模板(之前本来图省事，一直想选定Anisina的，但是后来发现还不如这个Hueman使用方便，并其实Hueman搭好了会更美观…)；然后又是git、Node、hexo的安装；加上调试git…

当时想部署到github上，hexo deploy，总提示\# ERROR Deployer not found: git，除了要在”_config.yml”中更改type字段为type: git，还要执行npm install hexo-deployer-git --save，每次新建一个hexo”project”都需要执行一次，好像是到了v3.0版本才这样的…

其中教程主要参考了“20分钟教你使用hexo搭建github博客”这篇简书，还是很有帮助的…以后估计也还会需要查阅…

总之，有个能看的页面了，接下来需要把图片的问题搞定了，毕竟Hueman的模板是有图才好看~ 以后有时间再鼓捣吧…因为再不开始干活，就要过年了…\ue412

看了一下表，该去吃晚饭了…估计还是一碗泡面吧…\ue412
